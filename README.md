# addis-react

### A simple crud app created using <br>
  -> react <br>
  -> react-redux <br>
  -> redux-saga <br>


## to run the project

### install dependencies
```npm install```

### run the project
```npm run dev```

### initialize the api
on a separate terminal run

```json-server --watch db/db.json --port 8000```

### open your browser and go to the url "localhost:3000"
